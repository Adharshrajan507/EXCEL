The Football Tournament Dashboard is an interactive Excel dashboard that provides a comprehensive overview of football tournament statistics across different seasons and competitions. Using dynamic slicers, users can filter the data to view key insights such as the Top Scorer, Top Scoring Team, Top 10 Scorers, Top 10 Assist Providers, Card Distribution Summary, Match Outcome Analysis, and Player Stat Analysis. The dashboard enables users to compare team and player performances, identify top performers, and analyze tournament trends, making it a valuable tool for football performance analysis and data-driven decision-making.
