The Sales Distribution Dashboard is an interactive Excel dashboard that provides a concise overview of sales performance. It allows users to analyze data by filtering Month and Region, while displaying key metrics such as Total Sales, Monthly Target vs. Actual Sales, Top 3 Customers, Top 10 Products, Regional Sales Distribution, Weekly Sales Trends, and Country-wise Sales. The dashboard helps users quickly identify sales patterns, top performers, and regional performance, enabling better business analysis and decision-making.
